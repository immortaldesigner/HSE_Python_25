# Homework 2. TG Bot

** Доступен в telegram @TEST_HW_CALORIES1_BOT **

ВНИМАНИЕ: В этой работе использовалась генеративная модель в следующих местах:
  - Создание Юникод Эмоджи для более красивого пользовательского интерфейса
  - Создание requirements.txt для выбора правильных версий
  - Вся логика и остальной код был написан руками

Функции:
1. Динамичесикая анкета с редактированием и удалением мусора через кнопки.
  - Вес
  - Рост
  - Возврат
  - Уровень активности
  - Город
  - Сохранение данных в users.json
2. Еда
  - Добавление еды по названию
  - Добавление еды по штрихкоду
  - Добавление еды по фото (на фото должен быть виден штрихкод)
  - Влияние еды на ежедневную цель
  - Расчет калорий исходя из вида и массы продукта
3. Вода
  - Расчет нормы воды исходя из данных анкеты
  - Добавление воды
  - Отображеие истории
4. Тренировки пяти видов
  - Тренировки учитывают потребление воды исходя из температуры
  - Тренировки дома будут брать среднюю температуру в комнате
  - Тренеровки на улице будут брать температуру с api OpenWeather
  - Влияние тренировок на ежеденую норму калорий и воды
5. Цели
  - Ежедневная норма расчитается по улучшенной формуле исходя из уровня активности
  - Учитывание еды и воды
  - Напоминания о ежедневных тренировках с регулируемым таймером
6. Прогресс
  - Визуализация графиков
7. Дополнительно
  - Логирование всех сообщений
  - Автоматические сохранение при завершении работы бота
  - Исправлени ошибок при вводе
